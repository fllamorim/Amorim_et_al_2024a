# Tipping of the double-diffusive regime in the southern Adriatic Pit in 2017 in connection with record high-salinity values

This project analyses potential temperature and salinity data from mooring and Argo floats, calculates the Turner angle and vector length.

## Tools
Python
Pandas
Xarray
Matplotlib
numpy
cartopy

## Datasets
- EMSO-E2M3A temperature and salinity (http://emso.eu/observatories-node/south-adriatic-sea/)
- Argo float number 6903197 profiles (https://www.euro-argo.eu/Argo-Data-access)
- ERA5 net surface heat fluxes and mixed layer depth
- Surface geostrophic velocities (SEALEVEL_EUR_PHY_L4_MY_008_068 product; Copernicus Marine Service)
- 
## Analysis
- Data preprocessing
- Time series and profile analysis
- Double diffusion

## Key Findings
- Strong heat loss at the surface during winter allowed deep convection to transport high-salinity water from the intermediate to the deep layers
- Increased predisposition to slt fingers throughout the water column.
- Alteration of vertical stratification throughout the water column during the winter of 2016/17 with no return in the following years
